Parking App

How to use the application:

Wait for the flying pigs launch screen

Click the Clean button to clean your source code.

Some useful features:

  1. Add Parking
  2. View Parking Receipt
  3. View Parking Receipt List
  4. Search Nearby Parking Facilities
  5. Update User Profile
  6. SignOut
